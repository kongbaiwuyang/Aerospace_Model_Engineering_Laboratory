# Aerospace_Model_Engineering_Laboratory
航模工程实验室 V4.0
=====================

简介
----
航模工程实验室是一个综合性的航模设计与分析工具，集成了空气动力学计算、3D建模、动力系统仿真和稳定性分析等功能。该工具帮助用户快速评估航模设计方案的性能，包括失速速度、推重比、稳定性等关键指标。

主要特性
--------
1. 实时3D飞机建模
   - 完整的飞机外壳（机身、机翼、尾翼、螺旋桨）
   - 动态重心和中性点标记
   - 可旋转、缩放的3D视图

2. 全面的气动分析
   - 失速速度计算
   - 静态推力和推重比分析
   - 雷诺数和流态分析
   - 静稳定性裕度计算

3. 动力系统优化
   - 电机KV值和内阻匹配
   - 电池配置分析
   - 螺旋桨参数（直径、螺距、叶片数）优化
   - 效率和续航时间预测

4. 环境适应性
   - 标准大气模型（ISA）
   - 海拔高度修正
   - 温度影响分析

5. 可视化设计
   - 2D平面设计图
   - 关键参数仪表盘
   - 实时计算日志

安装说明
----------
直接在现代浏览器中打开 index.html 文件即可使用（推荐Chrome/Firefox最新版）

所需依赖
----------
- Three.js (在线CDN加载)
- HTML5 Canvas
- JavaScript ES6+

使用方法
--------
1. 在左侧参数面板中输入您的设计参数
2. 实时查看3D模型变化和分析结果
3. 观察右侧仪表盘中的关键性能指标
4. 查看底部日志窗口了解详细分析过程

参数说明
--------
环境参数：
- 海拔高度：0-5000m
- 环境温度：-40°C到50°C

气动几何：
- 翼展：100-3000mm
- 根弦/梢弦：20-500mm
- 起飞重量：50-5000g
- 平尾面积/力臂：0.1-10dm² / 50-1500mm

重心位置：
- 重心位置：10%-40%MAC

动力系统：
- 电机KV值：100-3000
- 内阻：0.01-0.5Ω
- 电池配置：2S-6S
- 螺旋桨：直径3-20英寸，螺距1-15英寸，2-6叶片

分析结果
--------
- 空气密度：kg/m³
- 失速速度：km/h
- 推力和推重比
- 全速电流和续航时间
- 静稳定裕度
- 深度分析（雷诺数、升力线斜率等）

安全提示
--------
- 注意螺旋桨转速（桨尖马赫数应小于0.7）
- 确保静稳定裕度在5%-25%范围内
- 电池电流不超过放电倍率限制
- 避免过高的推重比导致操控困难
